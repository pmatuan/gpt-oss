public class EchoRequest {

```java

## Exercise 3: Using @PathVariable

Question: Write a Spring Boot REST controller with a GET endpoint "/square/{number}" that takes a path variable "number" and returns the square of that number as a string.

Answer:

```java
import org.springframework.web.bind.annotation.GetMapping;
import org.springframework.web.bind.annotation.PathVariable;
import org.springframework.web.bind.annotation.RestController;

@RestController
public class MathController {
    
    @GetMapping("/square/{number}")
    public String square(@PathVariable int number) {
        int square = number * number;
        return String.valueOf(square);
    }
}

import codecs

### 4. 在Java中使用 `InputStreamReader` 和 `OutputStreamWriter`

在Java中，你可以使用 `InputStreamReader` 和 `OutputStreamWriter` 来处理不同编码的文件：

```java
import java.io.*;

public class EncodingExample {
    public static void main(String[] args) throws IOException {
        // 读取GBK编码的文件
        try (BufferedReader reader = new BufferedReader(new InputStreamReader(new FileInputStream("input.txt"), "GBK"))) {
            String line;
            try (BufferedWriter writer = new BufferedWriter(new OutputStreamWriter(new FileOutputStream("output.txt"), "UTF-8"))) {
                while ((line = reader.readLine()) != null) {
                    writer.write(line);
                    writer.newLine();
                }
            }
        }
    }
}

**Answer**:  
Photosynthesis is how plants make food.  
- Plants take in sunlight, water from the soil, and carbon dioxide from the air.  
- Inside the plant’s leaves, a green pigment called chlorophyll captures the sunlight.  
- The plant uses the sunlight’s energy to mix the water and carbon dioxide into glucose (a type of sugar) and oxygen.  
- The glucose is used as food for the plant, and the oxygen is released into the air for us to breathe.

**Question 4**  
**Prompt**: Explain the process of photosynthesis in simple terms.  
**Answer**:  
Photosynthesis is how plants make food.  
- Plants take in sunlight, water from the soil, and carbon dioxide from the air.  
- Inside the plant’s leaves, a green pigment called chlorophyll captures the sunlight.  
- The plant uses the sunlight’s energy to mix the water and carbon dioxide into glucose (a type of sugar) and oxygen.  
- The glucose is used as food for the plant, and the oxygen is released into the air for us to breathe.
